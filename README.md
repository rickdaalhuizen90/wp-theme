# Wordpress Starter Theme

## Get Started
```
npm install
```

## Watch CSS
```
npm run css:watch
```

## Code conventions
- CSS: [https://stylelint.io/](https://stylelint.io/)
- JS: [https://eslint.org/](https://eslint.org/)

Run linter
```
npm run lint:css
npm run lint:js
```

## Build js
```
gulp js:build
```
